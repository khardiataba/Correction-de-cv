# Correction-de-cv
Ce projet consiste à analyser et corriger un CV pour le rendre professionnel et percutant.

📝 Objectifs
Identifier les 5 erreurs majeures du CV original.

Proposer des corrections concrètes (verbes d'action, chiffres, structure).

Produire une version finale optimisée et prête à l'emploi.

🛠️ Travail Réalisé
Analyse : Repérage des points faibles (flou, manque de résultats, design).

Correction : Réécriture des expériences et réorganisation des compétences.

Livrable : Version corrigée disponible en téléchargement.
